# Projeto 1 da Disciplina de Computação Gráfica

Implementação de uma cena do desenho animado Bob Esponja. Todos os objetos foram implementados de forma manual e utilizando apenas Python e OpenGL.

## Interação com o usuário:
- Tecla P: Exibe a malha poligonal da cena
- Tecla UP: Movimenta o personagem Patrick para frente
- Tecla LEFT: Rotaciona o personagem Patrick no sentido horário
- Tecla RIGHT: Rotaciona o personagem Patrick no sentido anti-horário
- Tecla A: Desce o Anzol 
- Tecla Q: Sobe o Anzol
- Tecla X: Aumenta o personagem Bob Esponja
- Tecla Z: Diminui o personagem Bob Esponja
